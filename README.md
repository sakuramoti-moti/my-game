# 猫のアクションゲーム - ネズミ復活版 🐱🐭

ブラウザで遊べる、シンプルで爽快な横スクロールアクションゲームです。
可愛い猫を操作して、敵のネズミを倒しながらゴール（鳥居）を目指しましょう！

## 🎮 ゲームの遊び方

### 操作方法
- **PCの場合**
  - **◀ / ▶ キー**: 左右移動
  - **Space キー**: ジャンプ
- **スマートフォンの場合**
  - 画面上のボタン（◀ / ▶ / JUMP）をタップして操作

### ルール
- **小判（金色の円）**: 拾うとスコアが加算されます。
- **ネズミ**: 踏みつけると倒せます（スコア加算）。横からぶつかるとミスになり、ステージの最初に戻ります。
- **トゲ**: 触れるとミスになり、ステージの最初に戻ります。
- **鳥居（⛩️）**: ゴールです！触れるとステージクリアとなり、次のステージへ進みます。

## ✨ 特徴
- **インストール不要**: HTMLファイル一つで動くので、ブラウザがあればどこでも遊べます。
- **レスポンシブ対応**: PCでもスマホでも快適にプレイ可能です。
- **全画面モード**: 右上の「全画面」ボタンで没入感のあるプレイが楽しめます。

## 🛠️ 技術スタック
- **HTML5 Canvas**: グラフィック描画
- **JavaScript (Vanilla JS)**: ゲームロジック
- **Web Audio API**: レトロなBGMと効果音
